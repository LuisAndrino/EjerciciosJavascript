Ejercicio 20

Mediante un script el usuario debera teclear una letra y numero y el programa mostrara una cadena formada por la letra tecleada y el numero de veces que se repita la letra. Por ejemplo, si tecleo la letra "a" y el numero 5, el programa mostrara la cadena "aaaaa".

        Codigo:
        
            let ingreseLetra = prompt("Ingrese una letra");
            let ingreseNumero = prompt("Ingrese un numero");
            let resultado = "";

            for(let i = 0 ; i < ingreseNumero ; i++){
                resultado += ingreseLetra;
            }

            alert(resultado);
