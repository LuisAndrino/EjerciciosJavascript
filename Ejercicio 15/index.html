Ejercicio 15

Escriba un programa que pregunte una y otra vez si se quiere seguir ejecutando el programa, siempre que se conteste sí. (en minuscula y con tilde); 


        let condiciones = true;
        let pregunta = prompt("Quiere seguir ejecutando el programa? conteste con un sí");
        while(condiciones){
            if(pregunta == "sí"){
                pregunta = prompt("Quiere seguir ejecutando el programa? conteste con un sí");
            }else{
                condiciones = false;
            }
        }
