Ejercicio 4

Elaborar un programa que solicite al usuario que teclee un numero entre 1,2,3, y 4 . Si pulso 2 mostrara en consola el mensaje que ha acertado, en caso contrario dira que se equivoco


        let numero = prompt("Elija un numero entre el 1 y el 4");

        numero = parseInt(numero);

        if(numero == 2){
        alert("Felicidades, acertaste");
        } else if (numero == 1 || numero == 3 || numero == 4){
        alert("Fallase :(")
        } else {
        alert("Opcion invalida");
        }
