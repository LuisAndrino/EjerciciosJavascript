Ejercicio 12

En un restaurante los clientes pueden pedir carne, pescado o verduras. Si pide carne se le ofrecera como bebida vino tinto, si pide pescado se le ofrecera como bebida vino blanco y si pide verduras se le ofrecera como bebida agua.

Si no elije una opcion del menu, aparecera la opcion de elija carne, pescado o verduras


        let eleccion = prompt("Ingrese una opción: carne, pescado o verduras");

        switch (eleccion) {
            case "carne":
                alert("Elegiste carne, dejame recomendarte vino tinto como bebida");
                break;
            case "pescado":
                alert("Elegiste pescado, dejame recomendarte vino blanco como bebida");
                break;
            case "verduras":
                alert("Elegiste verduras, dejame recomendarte agua como bebida");
                break;
            default:
                alert("No elegiste una opción válida, elige entre carne, pescado o verduras");
                break;
        }
